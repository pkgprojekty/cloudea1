# CloudEA - Strategy & Architecture

Production-ready Next.js package for Vercel deployment.

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this folder to GitHub or drag-and-drop it into Vercel.
2. Vercel will detect Next.js automatically.
3. Click Deploy.

## Included
- Alysian-style consulting layout
- PL / EN language switch
- Updated contact details
- Business Capability Map, architecture maturity, and crafted projects sections
- Next.js 14.2.35
